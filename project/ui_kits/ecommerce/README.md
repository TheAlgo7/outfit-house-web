# UI Kit · The Outfit House Storefront

A high-fidelity React (JSX-via-Babel) recreation of the e-commerce
storefront. Click-thru navigation between **Home → Product → Cart**.

## Files
- `index.html` — interactive demo entry
- `Header.jsx` · `Footer.jsx` · `Buttons.jsx`
- `ProductCard.jsx` · `ProductGrid.jsx`
- `ProductDetail.jsx` · `Cart.jsx`
- `data.js` — product catalog (placeholder)

All components consume CSS variables from `../../colors_and_type.css`.
