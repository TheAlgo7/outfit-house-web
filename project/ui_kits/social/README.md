# UI Kit · Instagram Post Templates

Square (1080×1080) social post templates that match the brand's matte-black,
warm-neutral identity. Recreates the visual language of the supplied
`insta-post-tote.jpg` reference.

## Files
- `index.html` — three sample posts side-by-side
- `PostFrame.jsx` — base 1080² frame with brand chrome (logo, footer)
- Variants embedded in index: Product Drop, Quality Spec, Brand Hero
