## Quick reference

- **Brand:** The Outfit House — premium 7A/UA streetwear & sneaker store, New Delhi.
- **Tone:** confident, technical, understated. Uppercase headings. No emoji in UI.
- **Palette:** `#000` black · `#F2ECDD` eggshell · `#E8D3B2` hampton · `#E3A848` Indian Yellow (accent only).
- **Type:** Bebas Neue display · Archivo headings · Inter body · Permanent Marker brush (marketing only).
- **Tokens:** import `colors_and_type.css` to get every CSS var + base type styles.
- **Components:** see `preview/` cards and `ui_kits/ecommerce/` + `ui_kits/social/`.
- **Iconography:** Lucide via CDN, 1.5 stroke; the crown PNG (`assets/logo-crown.png`) is the brand mark.
