# SEO Action Plan — The Outfit House
**theoutfithouse.in** · Updated 2026-06-17 · Score: 68 → ~88 (projected) · Status: implemented

The full action plan from the 2026-06-17 audit has been implemented in one pass. This
file now records what shipped and the short list of things only verifiable after deploy.

---

## ✅ Done

### Critical
- **CRIT-1 — Catalog pre-rendered.** `build.js` generates a fully static, crawlable page for
  every product (`/product/<slug>/index.html`, 127) and category (`/<cat>/index.html`, 4) with
  baked title/meta/canonical/OG, body copy, static nav/footer, and JSON-LD. No JS needed to read them.
- **CRIT-2 — Brand-name purge.** All slugs and image filenames de-branded to silhouette+colour
  (e.g. `rolex-datejust` → `luxury-watch-silver-blue`; `AJ1 Mid Panda (1).webp` →
  `mid-top-sneaker-panda.webp`). 127 images renamed, `site.js` `PRODUCTS` rewritten, Product.html
  fallback brand-map removed, homepage hardcoded paths fixed. 127 × 301 redirects added.
  Verified: **zero brand tokens** across the deployable surface.
- **CRIT-3 — Sitemap rebuilt.** 137 URLs (all 127 products + 4 categories + 6 core pages), fresh
  `lastmod`, de-branded image entries. Auto-regenerated by `build.js`.

### High
- **HIGH-1 — Unique static titles/meta** on all 127 products + 4 categories (baked at build).
- **HIGH-2 — Static nav + footer** inlined into every generated page (real internal link graph).
- **HIGH-3 — Schema accuracy.** `availability` now per stock flag (`InStock` for held, `BackOrder`
  for arrange-on-request); false `ClothingStore` 12–21 daily hours removed (store is pre-launch).

### Medium
- **MED-1** — `llms.txt` `/policy` → `/terms`; added `/guide`.
- **MED-2** — `og:image:width/height` (1200×630) on all pages.
- **MED-3** — Google Fonts made non-blocking (`media="print"` swap + `<noscript>` fallback) everywhere.
- **MED-4** — Unique, brand-free description per product (silhouette + tier build language).
- **MED-5** — Richer product alt text (silhouette + colour + tier).

### Low
- **LOW-1** — Removed 50 stray brand-named source files (`.jpg` + orphan `.png`); `assets/` 60 MB → 27 MB.
  Added `.vercelignore` + robots/redirect blocks so docs/build files never deploy or get indexed.
- **LOW-2** — Image sitemap now covers all 127 products.
- **LOW-3** — Buying-guide page (`/guide`) with `Article` + `FAQPage` schema, wired into nav/footer/sitemap/llms.
- **LOW-4** — `www` → apex 301 added to `vercel.json` (confirm domain is attached in Vercel too).
